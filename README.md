# Flask digital library

A simple user authentication system built with Flask and SQLAlchemy.

This project allows users to:

- Register an account

- Log in with credentials

- Store user data in a database

Access the main page after authentication
---

### Features

- User registration

- Login system

- SQLite database

- Flask backend

- HTML templates

- Technologies

- Python

- Flask

- Flask-SQLAlchemy

- SQLite

---

### Purpose

This project was created to practice:

Backend development

User authentication

Database integration

Web application structure

Small, simple, and focused on consistency.
